# typst-projects

Repositorio personal de documentos y plantillas hechas en [Typst](https://typst.app), principalmente formularios de apoyo para estudiar física (autoayuda propia). La idea es ir sumando más formularios a medida que avanzo en los cursos, y eventualmente extenderlo a otras ramas además de física.

## Estructura

- `main.typ` — documento principal (contenido del formulario).
- `config.typ` — configuración de layout (página, texto, párrafo, lista).
- `shortcuts.typ` — atajos de notación matemática (`vv`, `norm`, etc.).
